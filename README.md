## Installation

- Windows

```batch
git clone git@github.com:TsitouhRanjafy/git-follow-manager.git
cd git-follow-manager
npm ci 
```
run `node index.js -h` for help

- Based linux

```shell
git clone git@github.com:TsitouhRanjafy/git-follow-manager.git
cd git-follow-manager
npm ci
chmod +x index.js
```

run `./index.js -h` for help
